安全的用连接远程主机

VNC是一个连接系统,让您在本地操作远程服务器上的图形桌面,对于不熟悉命令行的用户,它使在远程服务器上管理文件和设置更加容易.以下操作将会在本地devuan系统和远程服务器上设置VNC服务器,并通过SSH隧道安全地连接服务器 在远程服务器上安装vnc服务软件
apt install tightvncserver
开启vnc服务
vncserver
设置密码
Password: Verify:
无说明
Would you like to enter a view-only password (y/n)? n
以下操作在本地主机进行.安装vnc软件
apt install gvncviewer
使用SSH安全转发到VNC的连接
ssh -L 5901:127.0.0.1:5901 -C -N -l user-name your_server_ip
开始连接,弹出一个窗口,输入第3步的密码
gvncviewer your_server_ip:1
