VNC是一个连接系统,让您在本地操作远程服务器上的图形桌面,对于不熟悉命
令行的用户,它使在远程服务器上管理文件和设置更加容易.以下操作将会在本
地devuan系统和远程服务器上设置VNC服务器,并通过SSH隧道安全地连接服务器


    

  1. 在远程服务器上安装vnc服务软件
    2. 

    
apt install tightvncserver


    

  2. 开启vnc服务
    3. 

    
vncserver


    

  3. 设置密码
    4. 

    
Password:
Verify:


    

  4. 无说明
    5. 

    
Would you like to enter a view-only password (y/n)? n


    

  5. 以下操作在本地主机进行.安装vnc软件
    6. 

    
apt install gvncviewer


    

  6. 使用SSH安全转发到VNC的连接
    7. 

    
ssh -L 5901:127.0.0.1:5901 -C -N -l user-name your_server_ip


    

  7. 开始连接,弹出一个窗口，输入第3步的密码
    8. 

    
gvncviewer your_server_ip:1




另外可以试试 ssvnc ,它为VNC连接增加了加密安全性,而且它还有GUI
