以前安装系统一般是把iso文件刻录到光盘,再从光驱启动进行安装.或刻录到
U盘启动安装.可是当你没有光驱也没有U盘怎么办呢?下面分3种情况来说说

(1)你的系统是2000/XP/2003

  1. 下载grub
    2. 

  2. 把里面的文件解压到C盘根目录
    3. 

  3. 编辑C:\boot.ini,把timeout的值改为大于0的数,在末尾添加一行代码
    4. 

    
  C:\grldr.mbr="GRUB MENU"
  
    

  4. 下载openbsd,放在C盘根目录里
    5. 

  5. 重启电脑.多了一项启动菜单,点进去就出现了openbsd--net-install
    6. 

  6. 选择它,开始安装有可能是世界上最安全的操作系统
    7. 


(2)你的系统是win7/8/10

  1. 下载grub
    2. 

  2. 把里面的文件解压到C盘根目录
    3. 

  3. 启动BOOTICE_x86_16.06.17_1.3.4.exe
    4. 



  4. 添加实模式启动项
    5. 



  5. 选择C盘所在的硬盘
    6. 



  6. 选择C盘
    7. 



  7. 依次点击"默认","保存当前系统设置","关闭"
    8. 



  8. 下载openbsd,放在C盘根目录里
    9. 

  9. 重启电脑.启动菜单多了一项Realmode App Loader,点击进去就出现了
openbsd--net-install,选择它,安装有可能是世界上最安全的操作系统
    10. 



(3)你的系统是Gnu/linux(debian/Ubuntu/Fedora)

  1. 编辑/etc/grub.d/40_custom,添加以下代码
    2. 

    
menuentry "openbsd--net-install"
 {
 linux16 /home/memdisk iso raw
 initrd16 /home/cd66.iso
 }
 
    


  2. 下载openbsd和memdisk,把它们放在/home/
    3. 

  3. 运行
    4. 

    
 grub-mkconfig -o /boot/grub/grub.cfg
 
    



  4. 重启电脑.启动菜单多了一项openbsd--net-install,选择它,安装有
可能是世界上最安全的操作系统
    5.
